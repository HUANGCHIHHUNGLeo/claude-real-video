# Security Policy

## Supported versions

The latest minor release line receives fixes. Older versions: please upgrade first and re-test.

| Version | Supported |
|---|---|
| 0.7.x (latest) | yes |
| < 0.7 | no |

## Reporting a vulnerability

Email **cortexos.main@gmail.com** with a description of the issue and steps to reproduce. Please do not open a public issue for security-sensitive reports.

You will get a reply on a best-effort basis (this is a one-person project); confirmed issues are fixed in the next release with credit if you want it.
